1

Create a private GitHub repository

Create `pennyspawn-private-engine`, select Private, open a Codespace, and upload the ZIP from this chat. Do not upload it into this public Pages repository.

2

Add GitHub Actions secrets

Inside the private repository, add CLOUDFLARE_API_TOKEN, CLOUDFLARE_ACCOUNT_ID, and ADMIN_TOKEN. Add PAY_TO as a repository variable containing only your public 0x receiving address.

Never add a seed phrase or private key.
3

Run testnet first

Actions → Deploy Private PennySpawn Q-Engine → Run workflow → testnet. Then paste the workers.dev URL into this dashboard.

The system can meter paid jobs, mutate strategies, retire failed generations, and create logical offspring. It cannot guarantee traffic, customers, $0.50 in one hour, or profit. Crypto payments may create tax and reporting obligations.
