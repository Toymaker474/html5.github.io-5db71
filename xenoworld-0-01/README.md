# PROJECT XENOWORLD 0.01

A mobile-first, offline-capable WebGL2 alien ecosystem sandbox. The playable simulation is contained in `xenoworld_0_01.html`; the manifest, icon, and service worker only provide Safari Home Screen installation and reliable offline launch after the first visit.

## iPhone / iPad install

1. Open the GitHub Pages project URL in Safari.
2. Tap **Share**.
3. Tap **Add to Home Screen**.
4. Launch **Xenoworld** from the new icon once the first page load has completed.

## Tests

Open `xenoworld_0_01.html?test=1` to run 20 deterministic contract checks.

## Honest v0.01 scope

Implemented: procedural WebGL2 planet and terrain, physical plant lifecycle, simple procedural body graphs and meshes, utility AI, memory, drives, signals, reproduction, mutation, death, decomposition, weather, world tools, local saves, and deterministic tests.

Roadmap: multiple planets, full chemistry, advanced soft-body dynamics, persistent civilizations, full language evolution, and large-scale chunk streaming.
